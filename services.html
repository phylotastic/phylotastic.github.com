The Phylotastic project has created several web-services. Below are the services that make up the phylotastic ecology of web services. You can find more information on these services in our Github repository.

  • Forester Web Service
  • WSExecution Web Service
  • User Case : Build Tree From URL Web Service
  • User Case : Build Tree From Text Web Service
  • User Case : Build Gene Species Reconcilation Tree Web Service
  • Web Service to Build Tree based on URL or Text (newick, nexml, phyloxml)
